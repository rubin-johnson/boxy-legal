Boxy — Privacy Policy

Last updated: April 2026

Overview

Boxy is a private personal system operated by a single individual for their own use. This policy describes how data is collected and handled.

What We Collect

Boxy may collect the following data in the course of normal operation:

  • Inbound and outbound SMS message content
  • Call metadata (timestamps, duration, phone numbers)
  • Personal health and task data entered by the system owner
  • System usage and performance metrics

How Data Is Used

All data is used solely for the operation of the personal Boxy system. Data is not used for advertising, analytics, or any commercial purpose.

Data Sharing

Data collected by Boxy is not sold, rented, or shared with any third party, except as required to operate the service (e.g., message transmission via Twilio, cloud infrastructure via Amazon Web Services). These providers are bound by their own privacy policies and are not granted permission to use data for any purpose beyond service delivery.

Data Storage

Data is stored on private cloud infrastructure (Amazon Web Services, us-east-1 region) controlled by the system owner. Sensitive data is encrypted at rest.
