Boxy — Privacy Policy

Last updated: May 15, 2026

Operator

Boxy is a private personal assistant system created and operated by Rubin Johnson, an individual, for their own private use. Rubin Johnson is the sole operator and sole recipient of all messages sent by this system.

Overview

This policy describes how data is collected and handled by the Boxy system operated by Rubin Johnson.

Mobile Opt-In and Consent

Rubin Johnson, as the system operator and sole recipient, activated SMS notifications by texting the keyword START to the Boxy toll-free number (+1-855-410-4957). No third parties are enrolled. Mobile opt-in consent data is not shared with, sold to, or transferred to any third party for any purpose.

Message frequency: approximately 10 messages per day. Message and data rates may apply. Text STOP to unsubscribe at any time. Text HELP for support.

What We Collect

Boxy may collect the following data in the course of normal operation:

  • Inbound and outbound SMS message content
  • Call metadata (timestamps, duration, phone numbers)
  • Personal health and task data entered by the system owner
  • System usage and performance metrics

How Data Is Used

All data is used solely for the operation of the personal Boxy system by Rubin Johnson. Data is not used for advertising, analytics, or any commercial purpose.

Data Sharing

Data collected by Boxy is not sold, rented, or shared with any third party for marketing, advertising, or any commercial purpose. Technical service providers (Twilio for message routing; Amazon Web Services for infrastructure) act solely as processors on behalf of Rubin Johnson under contractual obligations that prohibit any independent use of the data. These providers receive no opt-in consent data and have no independent rights over message content or recipient information.

Data Storage

Data is stored on private cloud infrastructure (Amazon Web Services, us-east-1 region) controlled by Rubin Johnson. Sensitive data is encrypted at rest.

Opt-Out

To stop receiving SMS messages, text STOP to the program number at any time. For further requests regarding your data, contact: rubin.lewis.johnson@gmail.com
