Boxy is a personal messaging relay and task management system operated for private use by its owner.

1. Program Description

Boxy sends SMS notifications related to personal task management, health reminders, calendar alerts, and system status updates. Messages are sent only to the registered owner of this system. This is not a marketing or promotional program.

2. How to Cancel

You can cancel SMS service at any time. Text STOP to the program number. After sending STOP, you will receive one confirmation message and no further messages will be sent. To restart, text START.
